# Estrutura Base - Loja de Materiais de Construção

Este projeto foi criado para servir como base para um sistema de vendas de materiais de construção. Ele utiliza:
- **Backend:** Node.js + Express + SQLite
- **Frontend:** React + JavaScript + HTML + CSS

## Funcionalidades implementadas
- Cadastro e listagem de produtos
- Cadastro e listagem de usuários
- Cadastro e listagem de pedidos

## Como expandir
- Adicione autenticação de usuários
- Implemente filtros e buscas
- Crie telas de administração
- Melhore o layout e responsividade
- Adicione testes automatizados
